
### The Equations:
1.  $2x + 3y = 12$
2.  $x - y = 1$

---

### 1. Isolate one variable
It is easiest to start with the second equation because the coefficients are simple. Let's solve for $x$:
$$x = y + 1$$

### 2. Substitute into the other equation
Now, we take this definition of $x$ and "plug it into" the first equation where the $x$ used to be:
$$2(y + 1) + 3y = 12$$

### 3. Solve for $y$
Now we have an equation with only one variable, which we can simplify:
* Distribute the $2$: $2y + 2 + 3y = 12$
* Combine like terms: $5y + 2 = 12$
* Subtract $2$ from both sides: $5y = 10$
* Divide by $5$: **$y = 2$**

### 4. Find $x$
Now that we know $y$ is $2$, we go back to our simplest equation ($x = y + 1$) to find $x$:
* $x = 2 + 1$
* **$x = 3$**
